# ip-geo

App for searching locations of IP addresses and URLs

## Installation

Clone this repo and npm install.

```bash
npm i
```

## Usage

### Development server

```bash
npm run dev
```

You can view the development server at `localhost:9000`.

### Production build

```bash
npm run build
```

### Lint

```bash
npm run lint
npm run lint:fix
```

### Testing

```bash
npm run test
```

### Coverage check

```bash
npm run test:coverage
```
